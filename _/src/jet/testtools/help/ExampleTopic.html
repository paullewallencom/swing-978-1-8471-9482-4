absolute value of
Calculates the absolute value of a number

For example, given the following case fragment,

Cholesterol     [0, 5.5]    7.3    5.6    6.1*
Triglyceride    [0, 1.8]    0.8    1.1    1.8
HDL             [0.9, 3.5]  2.2           0.5*

absolute value of ((previous to last HDL) - HDL) > 1.5

absolute value of (Cholesterol - Triglyceride) > 3

absolute value of (Cholesterol - upper reference value of Cholesterol) > .5
